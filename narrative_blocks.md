# Narrative Blocks Brainstorm (Home Page)

Goal: treat the homepage as a narrative landing page that still feels like a lively platform. The shelves are ambient (for immersion), while the narrative is carried by a few compact, platform-native blocks between shelf containers.

## Between-shelf narrative blocks (primary story spine)

1) **Start in Production (Phase 0)**
- Type: compact banner, full-bleed (outside shelf containers)
- Purpose: communicate “start imperfect, run in real workflows”
- Copy idea: “Deploy the starter stack into live workflows today. Learn from actual usage, not hypotheticals.”
- Placement: right after Hero

2) **Learning Window (Weeks 1–4)**
- Type: thin narrative strip with timeline chips
- Purpose: show early signal discovery and readiness
- Copy idea: “Week 1: adoption signals. Week 2: friction points. Week 3: automation value. Week 4: readiness map.”
- Placement: after first shelf group

3) **Stabilize + Adapt**
- Type: short “system evolution” banner
- Purpose: highlight iteration based on real usage
- Copy idea: “We modify agents, deepen integrations, and standardize what works.”
- Placement: mid-page, after integrations-focused shelves

4) **Bespoke Systems (End State)**
- Type: confident closing banner (still platform-native)
- Purpose: explain custom agents + bespoke workflows
- Copy idea: “From a starter stack to bespoke, production-grade systems built around your real operations.”
- Placement: near the end, before final shelves

## Optional narrative accents (still outside shelves)

5) **Timeline Rail**
- Type: thin vertical timeline on the left of the page content
- Purpose: anchor story progression across sections
- Copy idea: “Start → Learn → Stabilize → Bespoke”
- Placement: page-level UI element, subtle

6) **“This is not a demo” pledge**
- Type: short manifesto pill
- Purpose: reinforce production-first ethos
- Copy idea: “Real workflows first. Demos never decide the roadmap.”
- Placement: after Hero, before first shelf container

7) **“What changes after Week 4”**
- Type: small list card
- Purpose: summarize evolution phase
- Copy idea: “Agents re-scoped, integrations deepened, org-specific workflows defined.”
- Placement: after integrations-focused shelves
